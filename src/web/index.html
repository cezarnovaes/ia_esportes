Visão Geral do Projeto

Objetivo

Desenvolver um sistema de classificação de imagens esportivas utilizando técnicas avançadas de Deep Learning, especificamente Transfer Learning com MobileNetV2, para identificar automaticamente diferentes modalidades esportivas em fotografias.

Tecnologias Utilizadas

  • TensorFlow & Keras
  • MobileNetV2 (Transfer Learning)
  • Data Augmentation
  • Fine-tuning
  • Callbacks Avançados

MobileNetV2

Arquitetura eficiente otimizada para dispositivos móveis, utilizando depthwise separable convolutions para reduzir parâmetros mantendo alta performance.

Transfer Learning

Aproveitamento de conhecimento pré-treinado no ImageNet, congelando camadas iniciais e fine-tuning das últimas 20 camadas para adaptação ao domínio esportivo.

Regularização

Implementação de Dropout (0.2), BatchNormalization e Weight Decay para prevenir overfitting e melhorar generalização.

Metodologia de Desenvolvimento

Preparação dos Dados

# Configuração de Data Augmentation train_datagen = ImageDataGenerator( rescale=1./255, rotation_range=15, width_shift_range=0.15, height_shift_range=0.15, shear_range=0.15, zoom_range=0.15, horizontal_flip=True, brightness_range=[0.9, 1.1] )

Aplicação de técnicas de augmentação conservadoras para aumentar a diversidade dos dados de treino sem comprometer a qualidade das imagens.

Fine-tuning Strategy

  • Congelamento das primeiras camadas
  • Treinamento das últimas 20 camadas
  • Learning rate baixo (1e-4)
  • Otimizador Adam com weight decay

Callbacks Inteligentes

  • EarlyStopping (patience=10)
  • ReduceLROnPlateau (factor=0.5)
  • Restore best weights
  • Monitoramento de val_loss
